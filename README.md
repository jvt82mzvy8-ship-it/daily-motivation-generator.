# Daily Motivation Generator

A simple web application built with HTML, CSS, and JavaScript that displays a random motivational quote when the user clicks a button.

## Features

- Displays random motivational quotes
- Clean and responsive user interface
- Built with vanilla HTML, CSS, and JavaScript
- Beginner-friendly project

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Open `index.html` in your web browser.
2. Click the **Generate Quote** button.
3. A new motivational quote will appear each time you click.

## Purpose

This project was created as a practice exercise to demonstrate basic front-end web development and AI-assisted coding.
